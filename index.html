Rust Ray Tracer

  • Dragon is the 'Stanford dragon' commonly used to test 3D graphics software
  • Approx. 900,000 triangles
  • Uses a bounding volume hierarchy (BVH) for performance
  • Discuss things to note in image

Bounding Volume Hierarchy

  • Divides our triangles into a series of nested boxes, essentially creating a binary tree
  • Instead of searching for a ray intersection over all triangles we can simply walk the tree
